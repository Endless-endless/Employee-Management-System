# 职工管理系统（Employee Management System）

## 📌 项目简介

本项目是一个基于 C++ 开发的控制台职工管理系统，使用 Visual Studio 2022 编写，实现了对公司职工信息的基本管理功能。
项目主要用于练习 C++ 面向过程 / 面向对象编程思想、结构体（或类）的设计以及基本的数据管理逻辑。

---

## 🚀 功能介绍

系统主要实现以下功能：

* ➕ 添加职工信息
* ➖ 删除职工信息
* 🔍 查找职工信息
* ✏ 修改职工信息
* 📋 显示所有职工信息
* 🚪 退出管理系统

---

## ▶ 运行方式

1. 使用 Visual Studio 2022 打开 `.sln` 文件
2. 选择 Debug 或 Release 模式
3. 按 `Ctrl + F5` 运行程序

---

## 🧠 设计思路

* 使用类保存职工基本信息
* 通过菜单循环实现系统功能选择
* 通过数组 / 容器存储职工数据
* 使用函数封装各功能模块，提高代码可读性
